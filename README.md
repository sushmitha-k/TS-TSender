# 🚀 TSender

A **gas-efficient token airdrop dApp** built with modern web3 tooling.\
Send tokens to multiple recipients in a single transaction with an
optimized smart contract.

------------------------------------------------------------------------

## ✨ Features

-   ⚡ Gas-optimized airdrop contract (built in Huff)
-   🔗 Wallet connection via RainbowKit
-   🧾 Batch token transfers (multiple recipients)
-   🎯 Simple, clean UI for fast execution
-   🛠 Built with scalable frontend architecture

------------------------------------------------------------------------

## 🧱 Tech Stack

-   Framework: Next.js (App Router)
-   Styling: Tailwind CSS
-   Web3: Wagmi + RainbowKit
-   Smart Contracts: Huff
-   Language: TypeScript

------------------------------------------------------------------------

## 📦 Getting Started

### Clone the repo

git clone https://github.com/cyfrin/TSender.git\
cd TSender

### Install dependencies

npm install

### Run the development server

npm run dev

Open http://localhost:3000

------------------------------------------------------------------------

## 🧪 How It Works

1.  Connect your wallet\
2.  Enter token address, recipients, and amounts\
3.  Submit the form\
4.  Tokens are sent in one transaction

------------------------------------------------------------------------

## ⚠️ Notes

-   Double-check inputs before sending\
-   Transactions are irreversible
