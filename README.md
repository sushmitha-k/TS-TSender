# 🚀 TSender

A **gas-efficient token airdrop dApp** built with modern Web3 tooling.
TSender enables you to send tokens to multiple recipients in a single transaction using an optimized smart contract, reducing both cost and friction.

---

## ✨ Features

* ⚡ **Gas-Optimized Transfers**
  Built using Huff for ultra-efficient smart contract execution.

* 🔗 **Seamless Wallet Integration**
  Connect wallets بسهولة via RainbowKit.

* 🧾 **Batch Token Transfers**
  Send tokens to multiple recipients in a single transaction.

* 🎯 **Minimal & Intuitive UI**
  Designed for fast, distraction-free execution.

* 🛠 **Scalable Frontend Architecture**
  Built with maintainability and performance in mind.

## 📦 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/cyfrin/TSender.git
cd TSender
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Run Development Server

```bash
npm run dev
```

### 4. Open in Browser

```
http://localhost:3000
```

---

## 🧪 How It Works

1. Connect your wallet
2. Enter:

   * Token address
   * Recipient addresses
   * Token amounts
3. Submit the transaction
4. Tokens are distributed in a single on-chain transaction

---

## ⚠️ Important Notes

* Always **verify recipient addresses and amounts** before submitting
* Blockchain transactions are **irreversible**
* Ensure sufficient gas balance before execution
